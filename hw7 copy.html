Homework 7: ASL verbal morphology

Due: Monday, 3 December

This assignment is designed to use that skills of linguistics analysis we have learned throughout this course and apply them to a language in a different modality. At its core this is really a morphology problem, with a little bit of syntax thrown in. However, because ASL makes use of simultaneous (that is, nonconcatenative) morphology, different parts of verbal morphology are realized in a number of ways all in the same movement as the main verb. For each part you will be given a number of video clips representing one sentence in ASL, and then you will be given other, similar clips that you have to give translations for. Additionally there are a few questions asking you to make generalizations over the data that you have seen up until that part.

Technical details The videos should play in recent versions of Google Chrome, Internet Explorer (and mobile), Konqueror, Safari (and mobile), and the Android Browser. If you can't use one of these browsers, you should still be able to download each video file to your computer and play it back manually (with the caveat that you will have to make sure you match the names up to the sentences that are given on this webpage).
